# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added - MVP Release
- Backend FastAPI application with core endpoints
- Quality check API using YOLO for image analysis
- Price forecasting API with AI model
- Current price lookup with caching
- Price history and region comparison
- Frontend React application with Tailwind CSS
- Quality check UI with image upload
- Pricing dashboard with charts
- Docker Compose setup for easy deployment
- PostgreSQL database with models
- Redis caching layer
- Comprehensive documentation
- API documentation
- Quick start guide
- Contributing guidelines

### Features
- **Quality Check**: Upload images to analyze agricultural product quality
- **Price Forecast**: Predict prices for 7-30 days ahead
- **Current Pricing**: Get real-time price information by region
- **Price History**: View historical price trends
- **Region Comparison**: Compare prices across different regions

### Technical
- FastAPI backend with async support
- React + Vite frontend
- PostgreSQL 15 database
- Redis 7 for caching
- Docker containerization
- YOLOv8 for quality detection
- Scikit-learn for price forecasting
